# 📈 Stock Market Analysis - Reliance Industries

## Overview
End-to-end stock market analysis of Reliance Industries (NSE) using Python.
Data sourced live from Yahoo Finance API (2020–2024).

## Tools & Libraries
- Python, Pandas, Matplotlib, yfinance

## Key Analysis
- Closing price trend with 50-day & 200-day Moving Averages
- Daily Returns distribution & volatility measurement
- Trading Volume analysis over time

## Key Insights
- COVID-19 crash clearly visible in March 2020 (drop to ₹400)
- Golden Cross formed in mid-2020 — strong bullish signal
- Massive volume spike in July 2020 — Reliance Rights Issue (₹53,000 Cr)
- Stock recovered 3x from COVID low to ₹1300+ by 2024
